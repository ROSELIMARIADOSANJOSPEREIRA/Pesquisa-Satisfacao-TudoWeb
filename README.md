
# 📊 Pesquisa de Satisfação - TudoWeb

## 📝 Sobre o Projeto
Este é um script desenvolvido em Python para realizar pesquisas de satisfação de forma interativa. O programa foi criado para facilitar o levantamento de dados sobre o atendimento de uma empresa, permitindo que múltiplos usuários registrem suas notas.

## ⚙️ Funcionamento do Programa
O script segue uma lógica estruturada para garantir a integridade dos dados:
1. **Entrada de Dados:** Solicita o nome do cliente e a nota atribuída.
2. **Validação:** Verifica se a nota inserida está entre 1 e 10. Caso contrário, solicita uma nova entrada.
3. **Acúmulo:** O sistema armazena a soma das notas e conta o número de participantes.
4. **Condição de Parada:** O programa encerra a coleta quando o usuário digita o comando de saída estabelecido.
5. **Relatório Final:** Exibe a média aritmética de todas as notas coletadas.

1. Instale o [Python](https://www.python.org/) em seu computador.
2. Baixe o arquivo `pesquisa_satisfacao.py`.
3. Abra o terminal (ou CMD) na pasta do arquivo.
